# 대학교 공지 모아보기 서비스(frontend)

## 개발 목표
  1) 노션을 이용한 협업
  2) React 훅을 적극적으로 활용한 게시판 개발
     - componet 분리 및 활용
     - useEffect를 이용해 원하는 시점에서 함수 사용

## 개발 환경
  사용 언어: javascript(ES6)
  개발환경: React, Vite
